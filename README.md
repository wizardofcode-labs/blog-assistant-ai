# N블로그비서AI (ANBM)
블로그비서AI의 **공식 다운로드·자동업데이트 배포 채널**입니다.
설치파일은 이 저장소의 **[Releases](../../releases)** 에서 배포됩니다.

> 이 저장소는 배포 채널이며 **애플리케이션 소스코드는 포함하지 않습니다.**

## 다운로드
- 최신 버전: **[Releases 페이지](../../releases/latest)**
- Windows: `anbm-<버전>.exe` · macOS: `anbm-<버전>.dmg`

이전 버전이 필요하면 Releases 목록에서 해당 태그(`v1.3.0` 등)를 선택해 받을 수 있습니다.

## 시스템 요구사항
- Windows 10 이상 (64bit)
- macOS 11 (Big Sur) 이상

## 자동 업데이트
프로그램의 **[업데이트 확인]** 버튼을 누르면 최신 버전 여부를 확인합니다.
새 버전이 있으면 안내 후 최신 설치파일을 내려받아 설치할 수 있습니다.
